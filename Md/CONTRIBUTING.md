# Contributing to Smart Procurement Co-Pilot

Thank you for your interest in contributing! We welcome contributions from the community to make this Agentic AI system even better.

## How to Contribute

1.  **Fork the Repository**: Click the "Fork" button on the top right.
2.  **Clone your Fork**: `git clone https://github.com/YOUR_USERNAME/Smart-Procurement-Co-Pilot.git`
3.  **Create a Branch**: `git checkout -b feature/amazing-feature`
4.  **Make Changes**: Implement your feature or fix.
5.  **Test**: Run `python tests/automated_tests.py` to ensure nothing broke.
6.  **Commit**: `git commit -m "Add amazing feature"`
7.  **Push**: `git push origin feature/amazing-feature`
8.  **Open a Pull Request**: Submit your PR for review.

## Development Guidelines

*   **Code Style**: We follow PEP 8 for Python.
*   **Documentation**: Update `README.md` or `USAGE.md` if you change functionality.
*   **Tests**: Add a test case in `tests/` for any new logic.

## Reporting Issues

If you find a bug, please open an Issue with:
*   Steps to reproduce
*   Expected behavior
*   Actual behavior
*   Screenshots (if applicable)
